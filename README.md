## 🚀 Installation & Déploiement

### **Prérequis**
- Node.js >= 16
- npm ou yarn
- Git

### **Installation locale**

```bash
# Cloner le repo
git clone https://github.com/T0b0i7/New_Portefolio_2026.git
cd New_Portefolio_2026

# Installer les dépendances
npm install
# ou avec yarn
yarn install

# Lancer le serveur de développement
npm run dev
# ou
yarn dev

# Ouvrir dans le navigateur
# http://localhost:5173
```

### **Build pour production**

```bash
# Build optimisé
npm run build

# Prévisualiser la build
npm run preview

# Vérifier les erreurs
npm run lint
```

### **Déploiement**

#### **Sur Netlify (recommandé)**
```bash
# Install Netlify CLI
npm install -g netlify-cli

# Deploy
netlify deploy --prod

# Ou connecter votre repo GitHub pour auto-deploy
```

#### **Sur Vercel**
```bash
npm install -g vercel
vercel deploy --prod
```
